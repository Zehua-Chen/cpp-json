# Git

## Commit Messages

````
[type] Summary
Message
````

* `[type]`:
  * `[feat]`: add new feature;
  * `[refactor]`: refactor existing code;
  * `[doc]`: add documentation;
  * `[style]`: apply new visual style;
  * `[fix]`: fix issues;
  * `[test]`: add tests;
  * `[chore]`: add utility code (not included in production);
